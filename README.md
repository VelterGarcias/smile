# smile
<hr />

## Como começar?
1. git clone https://github.com/VelterGarcias/smile.git
2. npm install
3. npm run sass
4. começe a programar

## Como atualizar seu projeto antes de dar um git push ou de começar a programar?
```
git pull origin master
```
